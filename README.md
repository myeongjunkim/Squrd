## 구현 기능

### Account
- 로그인/로그아웃
- 회원가입
- 카카오톡 소셜 로그인
  

### Profile/instagram
- 로그인, 게시글 업로드, 댓글 업로드 할 경우 5포인트씩 누적
- 프로필 사진 등록 및 수정 

### Board
- 모달창을 통한 게시글 업로드
- 게시글별 디테일 페이지
- 게시글 삭제

### Vote
- 투표 등록, 이미지 클릭을 통한 선택
- 중복 투표 확인
- 댓글 및 대댓글 기능

### Crawling
- 총 10개의 언론사에서 각 페이지의 헤드라인 기사 크롤링
- 5분에 한번씩 크롤링하여 업데이트하는 스케줄링
- 썸네일, 기사제목, 기사링크 중심의 데이터 수집
